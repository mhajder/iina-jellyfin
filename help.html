Jellyfin Browser Sidebar

Use the "Show Jellyfin Browser" menu item (Cmd+Shift+J) or the sidebar to browse your Jellyfin server, search for media, and play URLs for playback directly in IINA.

Key features:

  • Direct media streaming from Jellyfin server
  • Browser-style interface for finding movies and TV shows
  • Automatic subtitle downloads with language preferences
  • Simple login/logout workflow

Getting Started

First Time Setup:

  1. Open IINA and go to Plugins â†’ Show Jellyfin Browser (or press Cmd+Shift+J)
  2. Click Connect to add your server
  3. Enter your Jellyfin server URL (e.g., http://192.168.1.100:8096)
  4. Enter your username and password
  5. Click Login to connect

Media Playback Workflow

The plugin provides direct media playback from the Jellyfin sidebar:

  1. Browse your recent items or search for content
  2. Click on a movie or episode
  3. The media opens directly in IINA with full video and audio
  4. Use preferences to choose between current window or new window
New Window Mode: Enable "Open media in new IINA window" in preferences to open each video in a separate window. This allows you to continue browsing while watching.
Logout: Click the Logout button to disconnect. You'll need to enter your server details again to reconnect.

Automatic Subtitle Downloads

The plugin automatically detects Jellyfin URLs and can download subtitles when enabled:

  • Jellyfin download URLs containing /Items/ and api_key=
  • URLs containing "jellyfin", "/Audio/", or "/Videos/"

Manual Actions

Available from the IINA menu bar:

  • Download Jellyfin Subtitles: Manually download subtitles for the current media
  • Set Jellyfin Title: Set the video title from Jellyfin metadata
  • Show Jellyfin Browser: Open the sidebar media browser (Cmd+Shift+J)
